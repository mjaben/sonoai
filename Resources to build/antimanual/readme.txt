== Description ==

**Antimanual is the all-in-one AI plugin for WordPress** — a single, modern dashboard that brings together a trainable AI chatbot, automated content publishing, bulk content rewriting, FAQ generator, documentation builder, repurpose studio, forum automation, AI-powered search, AI translation, SEO auditing, and intelligent internal linking.

Whether you run a SaaS product, an online store, a blog, or a community forum, Antimanual **replaces multiple plugins** with one tool and puts your most time-consuming site tasks on autopilot — no coding required.

**Use your own API key — we don't charge for it.** Antimanual works with your own OpenAI or Google Gemini API key. Both providers are supported in the free version, so you stay in full control of your AI usage and costs.

== Installation ==

1.  Upload the `antimanual` folder to `/wp-content/plugins/`, or install directly from the WordPress plugin repository.
2.  Activate the plugin through the **Plugins** menu in WordPress.
3.  Navigate to **Antimanual > Settings** in your dashboard.
4.  Enter your OpenAI API key to power the AI features.
5.  Add content to your **Knowledge Base** and enable the **Chatbot** — you are ready to go.

== Frequently Asked Questions ==

= What is Antimanual? =
Antimanual is an all-in-one AI WordPress plugin that automates the most time-consuming parts of running a website. It includes a trainable AI chatbot, automated content publishing, bulk content rewriting, FAQ generator, documentation builder, repurpose studio, forum automation, AI-powered search, AI translation (Pro), SEO auditing tools (SEO Plus), and an internal linking suite (SEO Plus) — all managed from a single, fast admin dashboard.

= Do I need an API key to use Antimanual? =
Yes, you need to provide your own API key from OpenAI or Google Gemini — we don't charge for API keys. Both OpenAI and Google Gemini are supported in the free version. Because you use your own key, you have full control over AI usage and costs — nothing is hidden or marked up.

= Which AI models does Antimanual support? =
The free version supports all current OpenAI models: GPT 5.4, GPT-5, GPT-4.1, GPT-4.1 Mini, GPT-4 Turbo, GPT-3.5 Turbo, and variants. Antimanual Pro adds Google Gemini models including Gemini 2.5 Pro, Gemini 2.5 Flash, Gemini 2.0 Flash, and more. The plugin validates your API key and shows only the models available to you.

= Can I train the chatbot on my own website content? =
Yes. The free version lets you add any WordPress post or page to the knowledge base with one click. The chatbot then uses AI vector embeddings of that content to give accurate, context-specific answers — not generic AI responses. Antimanual Pro extends the knowledge base to support PDFs, external website URLs, custom text, and GitHub repositories.

= Is auto-posting fully automated? =
Yes. Once you set up a schedule — choosing topics, tone, language, article length, and publishing time — Antimanual generates complete Gutenberg posts and publishes them automatically. The free plan supports up to 2 active schedules; Pro removes the cap entirely.

= What are the usage limits on the free version? =
The free plan includes: 60 AI chatbot conversations per month, 2 auto-posting schedules, 100 bulk rewrites per month, 100 forum AI answers per month, 100 forum topic conversions per month, and 100 lifetime AI search queries. Features with no usage limits on the free plan include: Generate Post with AI, Repurpose Studio, FAQ Generator, Docs Generator, and Auto Update. Upgrading to Pro removes all usage caps.

= What exactly is in the SEO Plus plan? =
SEO Plus includes everything in the free and Pro plans, plus two dedicated SEO modules. The SEO Agent runs AI-powered audits on any internal or external page, covers On-Page SEO, Technical SEO, Performance, Accessibility, Security, and Mobile Friendliness, and can auto-fix many common issues with a single click. The Internal Linking module provides a full link audit, AI link suggestions, bulk auto-linking, a visual site map, topic cluster builder, and anchor manager.

= How does Internal Linking work? =
The Internal Linking module (SEO Plus) analyses all posts and pages on your site to build a complete link report — showing total posts, orphan pages, inbound/outbound link counts, average links per post, and a Link Health Score. You can then generate AI-powered internal link suggestions for any post, apply links in bulk, visualise your entire site's link structure, and manage all anchor texts from one place.

= What languages does AI Translation support? =
Antimanual Pro includes AI translation for 34 languages. You can run bulk translations across all your posts, enable auto-translate on publish, add a custom glossary to protect brand terms, control tone and formality, and display a frontend language switcher that swaps content via AJAX — no page reload required. RTL languages are fully supported.

= Can I embed the chatbot on another website or domain? =
Yes. Cross-domain chatbot embedding via a simple iframe embed code is included with Antimanual Pro. This lets you deploy the same trained chatbot on multiple domains, documentation sites, or client properties — without any additional configuration.

= Does Antimanual integrate with community forum plugins? =
Yes. Antimanual integrates with [Forumax](https://wordpress.org/plugins/bbp-core) for AI-powered automatic responses to forum topics and replies. It also lets you convert closed forum threads into WordPress posts or knowledge base articles with a single click.

= Is my API key stored securely? =
Yes. All API keys are encrypted at rest using AES-256-CBC encryption tied to your WordPress secret keys. Keys are never sent to the browser unmasked — the admin UI shows only a masked preview with a connection status indicator.

= Can I use Antimanual with Google Gemini instead of OpenAI? =
Yes. Antimanual supports full Google Gemini for chat, knowledge base embeddings, and all content generation features. You can switch between OpenAI and Gemini providers at any time from the API Configuration page.

= Does Antimanual add any SEO markup automatically? =
The FAQ Generator adds structured FAQ Schema markup (JSON-LD) to FAQ blocks for Google rich results. The SEO Agent (SEO Plus) can auto-add missing meta descriptions, fix image alt text, add Open Graph tags, and enforce proper canonical URLs — all editable before saving.

== External Services ==

This plugin connects to the following external services.

= OpenAI API =

Powers AI chatbot, knowledge base embeddings, and all content generation features when OpenAI is the selected provider. Your WordPress post content (for KB indexing), visitor chat messages, and admin-configured prompts are sent to OpenAI via your own API key. Data is transmitted only when you trigger an AI action or a visitor interacts with the chatbot/search.

* [OpenAI](https://openai.com) — [Terms](https://openai.com/policies/terms-of-use) — [Privacy](https://openai.com/policies/privacy-policy)

= Google Gemini API =

An alternative AI provider (free plan) and the full AI backend in Antimanual Pro. The same categories of data described for OpenAI above are sent to Google Gemini via your own API key, only when Gemini is the active provider.

* [Google AI (Gemini)](https://ai.google.dev) — [Terms](https://ai.google.dev/gemini-api/terms) — [Privacy](https://policies.google.com/privacy)

= Spider Themes Notice Service =

The plugin periodically fetches an admin notice from Spider Themes servers to display important announcements to site administrators. Only a standard HTTP GET request is made — no personal data, site content, or visitor information is ever transmitted.

* [Spider Themes](https://spider-themes.net) — [Terms](https://spider-themes.net/terms) — [Privacy](https://spider-themes.net/privacy)
